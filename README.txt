== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `client-reviews-carousel.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Paste the shortcode [client-reviews-carousel] in the page you want to display the reviews
